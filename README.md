# Amazon-Customer-Review-Sales-
Novica Sales Data and Sentiment Analysis Project:

Developed an end-to-end NLP analytics pipeline processing Amazon customer reviews for 40,000+ artisan product SKUs to surface actionable customer insights. Implemented BERT-based sentiment analysis to quantify customer satisfaction at scale, and applied topic modeling using both LDA and NMF algorithms to extract recurring themes and pain points from unstructured feedback. Translated technical findings into business language through executive dashboards and professional visualizations, enabling the client to prioritize product improvements and refine customer experience strategy.
